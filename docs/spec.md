## One-liner
A daily-friendly emoji code-breaking puzzle: guess a hidden 4-emoji lock with clue feedback.

## Core loop
1. Player taps 4 emojis to form a guess.
2. Submit to receive clues: exact matches and misplaced matches.
3. Use clues to refine next guess.
4. Win by exact 4/4 before attempts run out.

## MVP scope
- Single puzzle mode (new puzzle button)
- 6-emoji pool
- 8 max attempts
- Guess history and feedback lines

## UX goals
- One-thumb operation
- Clear clue readability
- Fast replay (<10 sec to restart)
